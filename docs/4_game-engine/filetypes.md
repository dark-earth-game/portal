# File Types

## .TXT
Plain Text with certain formatting
* Dialogues
* Cinematics is Subtitles

## .CIF
Compressed Image File
* Font (7,8,11)
* JAP Font Glyphs
* Menu
* Inventory
* Scene Camera (View, Lightmap, Depth Buffer)
* Portraits
* Cinematics Stamps (thumbnails)


## .CTX
Context? / Compressed 


## Level
### .CIF Images
* F prefix - Image
* L prefix - Lightmap
* Z prefix - Depth buffer

### .MAP

Level (L/N prefixed files)
